# Concito / Klimaalliancen — Kommunernes Klimahandling

## Overblik

Concito (Danmarks Grønne Tænketank) driver i regi af **Klimaalliancen** (KL + Realdania + Concito + C40) et monitoreringsystem for alle danske kommuners klimaindsats. Den første statusrapport udkom april 2025 (209 sider).

**Kontaktperson**: Chefkonsulent Tue Damsø, CONCITO — ansvarlig for monitoreringsystemet og statusrapporten.

## Monitoreringsystemets fire hoveddele

1. **Rammerne for den kommunale klimaindsats** — kortlægning af indholdet i de 96 vedtagne klimahandlingsplaner (mål, delmål, tiltag)
2. **Status for fremdrift i kommunale klimatiltag** — spørgeskema ("Kommunernes årlige klimastatus") besvaret af 96 kommuner
3. **Omstillingsindikatorer for drivhusgasreduktion og klimatilpasning** — registerdata opgjort pr. kommune
4. **Tværkommunale projekter og Klimaalliancens aktiviteter** — samarbejdsprojekter

## Nøgletal fra rapporten (2022-data)

- **Samlet kommunal udledning**: 38 mio. ton CO₂e (2022), ned fra 47 mio. ton i 2018 (basisår) — ca. 20% reduktion
- **Sektorfordeling**: ~36% landbrug, ~30% energi, ~25% transport, rest affald/spildevand/processer
- **Kommunernes mål**: Reduktion til 18,8 mio. ton i 2030 (76% ift. 1990) — lineær fremskrivning giver 20,8 mio. ton
- **Prognose 2030** (tre scenarier):
  - Lineær fremskrivning: 20,8 mio. ton CO₂e
  - Eksponentiel udglatning: 21,4 mio. ton CO₂e
  - Konservativ prognose: 25,5 mio. ton CO₂e
- **35 kommuner** forventes at nå deres mål (lineær) — kun 18 i konservativt scenarie
- **96 af 98 kommuner** har vedtaget klimahandlingsplaner (DK2020-rammeværket)
- **Medianen** for klimaårsværk pr. kommune er 2 FTE (København: 123, Aarhus: 65)

## Geografisk variation

Concito-rapporten viser markante forskelle mellem kommunetyper:

- **Bykommuner** (København, Frederiksberg): Højt etageareal/indbygger, energisektoren dominerer udledning, størst absolut reduktion
- **Landkommuner**: Landbruget dominerer udledning, lavest/negativ reduktion, flest VE-potentialer
- **Industrikommuner** (Aalborg, Fredericia, Kalundborg): Stort industrielt energiforbrug
- **Ø-kommuner** (Læsø, Fanø, Samsø, Ærø): Lavt absolut forbrug, højt etageareal/indbygger

## Datakilder vi kan genbruge direkte

Flere af Concitos datakilder er offentligt tilgængelige registerdata:

### 1. Energi- og CO₂-regnskabet (SparEnergi)
- **Hvad**: Kommunefordelt drivhusgasregnskab, energiforbrug, VE-produktion
- **Kilde**: Ea Energianalyse for Energistyrelsen
- **URL**: https://sparenergi.dk/forbruger/vaerktoejer/energi-og-co2-regnskabet
- **Format**: Interaktivt dashboard — muligvis downloadbart som CSV
- **Opdatering**: Seneste data er 2022 (ca. 2 års forsinkelse)
- **Relevans**: Primær kilde til kommunefordelt CO₂-udledning — den mest direkte indikator for klimaindsats

### 2. Energidataservice
- **Hvad**: Detaljeret energiproduktion og -forbrug
- **URL**: https://www.energidataservice.dk/
- **Format**: REST API med JSON
- **Relevans**: VE-produktion pr. kommune (sol, vind)

### 3. DST BYGB40
- **Hvad**: Bygningsstatistik pr. kommune (etageareal, byggeår, type)
- **URL**: https://statistikbanken.dk/BYGB40
- **Format**: REST API (allerede brugt i projektet via fetch_dst.py)
- **Relevans**: Bygningsmasseindikator

### 4. Drivkraft Danmark / Evida
- **Hvad**: Oliefyr- og gasfyr-statistik pr. kommune
- **Relevans**: Opvarmningskilde-indikator
- **Begrænsning**: Ikke klart om data er frit tilgængeligt som download

### 5. Green Power Denmark
- **Hvad**: Vindmølle- og solcellekapacitet pr. kommune
- **URL**: https://www.greenpowerdenmark.dk/
- **Relevans**: VE-indikator — Klimabarometer-rapporten har kommunedata
