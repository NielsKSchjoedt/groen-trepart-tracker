# Glossary

## Danish Terms

| Term | English | Definition |
|------|---------|------------|
| Anlagt | Completed/constructed | Project status: physically built and operational |
| Basisanalyse | Baseline analysis | Initial environmental status assessment for a water body |
| Byråd | Municipal council | Top decision-making body of a municipality |
| Delopland | Sub-catchment | Subdivision of a main river catchment (108 total in Denmark) |
| Den Grønne Trepart | The Green Tripartite | Political agreement between government, agriculture, and food industry (June 2024) |
| Etablering | Establishment/construction | Project status: under construction |
| Forundersøgelse | Preliminary study | Project status: feasibility study in progress |
| Hovedvandopland | Main river catchment | Major drainage basin (23 in Denmark) |
| Indsats | Intervention/measure | A planned action to improve environmental status |
| Klima-lavbund | Climate-lowland | Carbon-rich lowland soil targeted for rewetting to reduce CO2 emissions |
| Kulstof2022 | Carbon2022 | 10×10m resolution map of carbon-rich lowland soils (118,000 ha identified) |
| Kvælstof | Nitrogen | Primary nutrient causing eutrophication in Danish coastal waters |
| Kystvandgruppe | Coastal water group | Geographic unit where nitrogen targets are measured (37 total) |
| Lavbundsjord | Lowland soil | Carbon-rich soil that emits CO2 when drained for agriculture |
| Lokal trepart | Local tripartite | Regional partnership organizing land transformation (23 total) |
| Miljømål | Environmental objective | WFD-required target for a water body |
| NKMv2025 | National Nitrogen Model v2025 | High-resolution model for nitrogen retention calculations |
| Omlægningsplan | Transformation plan | Municipal plan for converting agricultural land |
| Principbeslutning | Principle adoption | Municipal council decision to adopt a transformation plan |
| Skovrejsning | Afforestation | Planting new forest on previously non-forested land |
| Skitseprojekt | Sketch project | Early-stage project outline in MARS |
| Spærring | Barrier | Physical obstruction in watercourse (dam, weir, culvert) |
| Teknik- og miljøudvalg | Technical and environment committee | Municipal committee handling environmental matters |
| Tilsagn | Grant commitment | Formal commitment of subsidy funding for a project |
| Udtagning | Extraction/retirement | Removing agricultural land from production |
| Vandområdeplan | Water area plan | Water Framework Directive river basin management plan |
| Vådområde | Wetland | Area restored to wetland status for nitrogen removal |

## Technical Terms

| Term | Definition |
|------|------------|
| EPSG:25832 | ETRS89 / UTM zone 32N — standard coordinate system for Danish geodata |
| EPSG:4326 | WGS84 — global coordinate system used in web mapping |
| GeoJSON | RFC 7946 — JSON format for encoding geographic data structures |
| OGC | Open Geospatial Consortium — standards body for geospatial protocols |
| Retention | The fraction of nitrogen that is naturally removed (denitrified) between source and coast |
| Shapefile | ESRI format for geospatial vector data (.shp + .dbf + .shx + .prj) |
